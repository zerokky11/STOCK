스캔 설정

전체 유니버스를 넓게 훑으면서 초기 포착 후보를 올리는 기준입니다.

전체 탐지층이 최근 흐름을 비교할 시간 범위입니다.

값이 짧을수록 새로 강해지는 종목에 빨리 반응하고, 길수록 노이즈를 줄입니다.

초기 포착 점수가 이 수준을 넘으면 집중 감시 후보로 올립니다.

값이 높을수록 더 엄격하게 후보를 올리고, 낮을수록 넓게 관찰합니다.

아직 확정 신호 전 단계에서 최근 상승 속도가 살아나는지 보는 기준입니다.

장초반에는 조금 민감하게, 장중 이후에는 조금 더 엄격하게 두는 편이 자연스럽습니다.

평소보다 거래가 얼마나 빠르게 붙는지 먼저 보는 기준입니다.

값이 낮으면 초기에 더 많이 포착하고, 높이면 더 강한 거래 증가만 남습니다.

종목 크기 대비 자금 유입이 의미 있게 늘었는지 보는 기준입니다.

절대 거래대금보다 상대 강도를 먼저 보도록 만든 스캔용 기준입니다.

같은 섹터 안에서 먼저 움직이는 종목을 얼마나 빨리 후보로 볼지 정합니다.

값이 높을수록 섹터 동조 초입을 더 민감하게 잡고, 낮추면 개별 종목 강도를 더 우선합니다.

실시간 신호 설정

집중 감시 종목 안에서 실제 매수·매도 판단에 참고할 확정 신호 기준입니다.

최근 비교 구간 동안 이 이상 오르면 급등 후보로 봅니다.

값이 낮으면 신호가 많아지고, 높이면 더 강한 움직임만 남습니다.

급등 여부를 판단할 시간 범위입니다.

1분은 민감하고, 5분은 더 완만하게 봅니다. 장중 기본은 3분 기준을 권장합니다.

최근 거래량이 평소 대비 얼마나 늘었는지 보는 기준입니다.

값이 높을수록 평소보다 거래가 강하게 붙은 종목만 남습니다.

시가총액 대비 자금 유입 비율을 보는 기준입니다.

종목 크기 차이를 보정하는 데 도움이 됩니다. 절대 거래대금보다 상대 강도를 더 중심에 둡니다.

여러 조건을 합친 종합 점수의 컷라인입니다.

값이 높을수록 더 엄격한 신호만 보이고, 낮출수록 포착 범위가 넓어집니다.

초기 포착 이후 실제 돌파형 매수 신호를 얼마나 엄격하게 볼지 정합니다.

값이 높을수록 이미 힘이 충분히 확인된 돌파만 남기고, 낮추면 더 빠른 확인 신호를 허용합니다.

상승 흐름이 이어지는 종목을 실제 신호로 올릴 때의 엄격함입니다.

장중 이후에는 이 값을 조금 높여 추격 신호를 줄이고, 장초반에는 기본값으로 두는 편이 자연스럽습니다.

수익 보호용 매도 신호를 줄지 판단하는 기준입니다.

자동 주문 기준이 아니라, 수익이 일정 수준에 도달했을 때 익절 매도 신호를 줄지 보는 값입니다.

손실 관리용 매도 신호를 줄지 판단하는 기준입니다.

자동 손절 주문이 아니라, 손실이 기준을 넘었을 때 손실 관리 신호를 줄지 보는 값입니다.

상승 후 되밀림이 커졌을 때 수익 보호 신호를 줄 기준입니다.

자동 주문이 아니라, 고점 대비 밀림이 커질 때 추적손절 매도 신호를 줄지 판단하는 값입니다.

고급 설정

유동성이 너무 약한 종목만 걸러내는 보조 기준입니다.

메인 판단 기준은 RVOL, 최근 상승률, turnover_ratio입니다. 이 값은 절대 거래대금만으로 종목을 자르기보다, 체결 가능성이 너무 약한 경우만 막는 안전 하한으로 사용합니다.

초기 포착 후보를 한 번 더 걸러 주는 품질 기준입니다.

RVOL, 최근 상승률, turnover, 섹터 동조, 과열 리스크를 함께 반영한 점수입니다. 높이면 false positive 후보를 줄이는 데 도움이 됩니다.

후보가 websocket 집중 감시 대상으로 올라가기 위한 최종 기준입니다.

탐지 점수만이 아니라 후보 품질과 과열 리스크를 함께 반영한 우선순위 기준입니다.

후속 흐름이 없을 때 후보를 유지할 대략적인 시간입니다.

값을 짧게 두면 더 빠르게 후보를 정리하고, 길게 두면 한 번 더 후속 흐름을 기다립니다.

이미 늦은 추격 구간을 얼마나 강하게 감점할지 정합니다.

장 후반 급등이나 고가 바로 아래 급한 추격 후보를 줄이고 싶을 때 조금 높이는 편이 좋습니다.

윗꼬리와 되밀림 위험을 품질 점수에 얼마나 반영할지 정합니다.

값이 높을수록 장중 꼬리가 긴 후보를 더 보수적으로 해석합니다.

같은 종목이 최근 후속 흐름 없이 꺾였던 이력을 얼마나 감점할지 정합니다.

반복해서 false positive가 나오는 종목을 조금 더 뒤로 보내고 싶을 때 유용합니다.
